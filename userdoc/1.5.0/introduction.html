Introduction¶

Overview¶

FDO Toolbox is an application to view, manage and process spatial data. It is written in C# and uses the Feature Data objects (FDO) API. FDO Toolbox requires .NET Framework 2.0 or above to be installed.

FDO Toolbox is hosted on Github. You can download the latest binary releases and check out the latest source code using a git client.
